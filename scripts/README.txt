Scripts for molten salt fast reactor analysis
Ondrej Chvala, ochvala@utk.edu
GNU/GPL

This directory contains source code for MSFR analysis classes.

msfr.py   - builds and runs Serpent models of both the MSFR core and the wire irradiated by the depleted salt

agmsfr.py - analyzes the resuts of silver depletion, including resistivity changes

salts.py  - salt mixer

play*     - sandbox

See comments within the individual files for detailed code descriptions.

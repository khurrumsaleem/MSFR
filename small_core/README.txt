Here are scripts to run a specific analysis: to find and deplete the smallest spherical NaCl-UCl3 eutectic MCFR core with 19.75% LEU, and perform depletion analysis of silver in an MSFR environment. Each directory holds scripts relevant to a part of the analysis. 
Ondrej Chvala, ochvala@utk.edu
GNU/GPL

crit_search/	Searches for a critial dimension of the core.

refuel_search/	Searches for a refuel rate that keeps the core critical for a desired fuel cycle length. 

deplete_small/	Depletes a small core, scripts to plot results. 

eoc_results/	Plots results relevant to silver irradiation at the end of the cycle. 

old/		    Old stuff, ignore.
